**Basic Info.** This repo collects academic papers about **robot learning** for manipulation, control, and embodied AI. Papers are categorized by task and sorted by date (newest first). Papers with **open-sourced code** are marked with a star :star:.

Feel free to open a pull request for new papers/codes about robot learning.
